# Antigraviti Learning Base

このリポジトリは、**Antigraviti**（AIアシスタント機能を含む高度なシステム群）の概念、機能、そして実践的な活用方法を段階的に理解し、自身のスキルとして定着させるためのナレッジベースおよびプロジェクトリポジトリです。

AIとの対話を通じて得た知見や設定手法、ベストプラクティスを随時ドキュメント化し、蓄積していくことを目的としています。

## 🎯 目的 (Purpose)

- **Antigravitiの完全理解**: Workflows、Skills、MCPといった中核となる概念の仕組みを把握する。
- **実践的な活用スキルの習得**: Agent Managerを用いたプロンプトエンジニアリングや、各種ツールの効果的な連携手法を学ぶ。
- **ナレッジの体系化**: 対話プロセスで見つけた知見を蓄積し、後から振り返りやすいドキュメント（Knowledge Base）として整理する。
- **継続的な改善**: AIとのペアプログラミング・対話を通して、開発プロセスやタスク自動化の精度を継続的に向上させる。

## 📁 ディレクトリ構成 (Directory Structure)

現在の主要なディレクトリ構成は以下の通りです。

- `knowledge_base/` : Antigravitiに関する核となる知識をまとめたドキュメント群。
  - `00_core_concepts/` : Antigravitiを構成する基本概念（Workflows, Skills, MCPなど）の仕様や概要について。
  - `01_Agent_manager/` : Agent Managerの基本機能、Knowledgeの効果的な使い方、設定手順など。
  - `daily_reports/` : 日々の作業記録や振り返り、進捗状況のログ。
  - `000_document_formatting_guidelines.md` : ドキュメント作成時のフォーマットルール。
  - `progress_log.md` : プロジェクト全体の進捗管理ログ。

## 🚀 使い方・学習の進め方

1.  **AIとの対話・実験**: 気になる機能や設定についてAI（Antigraviti自身）と対話しながら試行錯誤します。
2.  **ナレッジの抽出**: 対話で得られた有益な情報、コードスニペット、設定例を抽出します。
3.  **ドキュメント化**: `knowledge_base` 以下の適切なディレクトリにMarkdown形式でまとめます。
4.  **リポジトリの更新**: 定期的にGitHubへPushし、学習の軌跡としてバージョン管理を行います。
