# 自動テスト実行とエラー解析

このワークフローは、プロジェクトのテストを実行し、その結果（特にエラー）をAIが自動で解析するためのものです。

## 1. テストの実行

プロジェクトに設定されている適切なテストコマンドを実行します。（例: アプリケーションの種類に応じて `npm test`, `pytest`, `flutter test` など）

// turbo

```bash
# 適切なテストコマンドを実行します
# 例: npm test
```

## 2. 結果の解析

- **すべて成功した場合**: ユーザーに「すべてのテストが成功しました。問題ありません！」と報告して終了します。
- **エラーが発生した場合**: AIはエラー出力（スタックトレースなど）を読み取り、以下の情報を分析します。
  - どのファイルの、何行目でエラーが起きているか
  - なぜエラーになったのか（原因の推測）
  - どう直せばよいか（具体的な修正コードの提案）

## 3. エラー報告と修正提案

エラーの分析結果をユーザーに提示し、「この修正案を適用しますか？」と確認を取ります。
