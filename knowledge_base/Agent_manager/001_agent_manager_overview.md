# Agent Manager の概要と基本機能

Agent Managerは、AIエージェントの能力を最大限に引き出し、開発やリサーチを効率化するための3つの主要機能（Playground、Knowledge、Browser）を提供する統括画面です。

## 1. 各機能の解説

### Playground（プレイグラウンド）

**「AIと自由に実験・テストができる砂場（サンドボックス）環境」**

メインのプロジェクトやソースコードを直接変更することなく、AIの挙動を試したり、特定のプロンプトのテストを行ったりするための独立した環境です。まさに「ブラウザで開くGemini（AIチャット）の出張所」のような役割を果たします。

- **主な用途**
  - **日常的な調べ物・リサーチ:** 開発環境から離れずに、翻訳、シンタックスの確認、ツールの概要調査などを手軽に行えます。
  - **プロンプトの調整・テスト:** 本番のコードを書かせる前に、意図した通りの回答やコードが生成されるかを試せます。
  - **メイン環境の保護:** プロジェクト本体とは関係ないタスクを切り離すことで、メインのチャット履歴やコンテキストをクリーンに保ちます。

### Knowledge（ナレッジ）

**「AIエージェントに専用の知識（コンテキスト）を与え、管理する場所」**

AIが回答を生成する際の「前提知識」となるドキュメントやルールラインを登録・管理するための機能です。

- **主な用途**
  - **プロジェクト固有のルールの参照:** コーディング規約、アーキテクチャのガイドライン、チーム内のルールなどをAIに共有します。
  - **外部ドキュメントのインポート:** 使用しているライブラリや独自APIの仕様書を読み込ませることで、より正確なコード生成を促します。
  - **コンテキストに沿った精度の高い回答:** 一般論ではなく、プロジェクトに特化した背景知識を基にAIが回答できるようになります。

### Browser（ブラウザ）

**「AIエージェントが自律的にWebを操作・視認するための統合ブラウザ」**

単なる表示用ではなく、**AI自身がWebブラウザを操作し、画面を「見る」ためのツール**です。人間のようにクリックやスクロール、情報の読み取りを行います。

- **主な用途**
  - **UI/UXの視覚的な確認:** 開発中のアプリケーションをAIに見せ、レイアウトの崩れ等を確認させます。
  - **Webからの情報収集:** 指定したWebサイトの最新情報を読み取らせ、内容を要約・抽出させます。
  - **ブラウザ操作のテスト自動化:** 実際の画面を操作しながらのエンドツーエンド（E2E）テストなどをAIに実行させることが可能です。

## 2. Playgroundのユニークで実践的な活用アイデア

Playgroundは「本番環境に影響を与えない独立した実験室」という特性を活かし、単なる調べ物以外にも様々な応用が可能です。

1. **「一時的なデータ生成」工場として使う**
   - 例: 「ECサイトのテスト用に、架空の商品名・価格・カテゴリが入ったJSONデータを50件生成して」
   - メリット: メインチャットを長大なデータで埋めることなく、必要な部分だけをコピーして利用できる。

2. **コードの「翻訳・別言語への変換」シミュレーター**
   - 例: 「このPythonのスクリプトをRustで書き直すとどうなる？ パフォーマンスの違いも教えて」
   - メリット: 実際のプロジェクトに影響を与えずに、別言語の書き方や設計思想を学べる。

3. **正規表現や難解なSQLの「サンドボックス」**
   - 例: 「このテキスト群から『2024年以降の日付』だけを抽出する正規表現を書き、テストケースで検証して」
   - メリット: 実際のコードに組み込む前に、エッジケースの漏れがないか安全にテスト・調整ができる。

4. **ペルソナ（人格）を変えた「レビューアー」として使う**
   - 例: 「あなたは厳格なセキュリティ専門のシニアエンジニアです。以下のコードの脆弱性を厳しく指摘してください」
   - メリット: 普段の優しいアシスタントとは違う厳しい視点でのレビューを引き出し、コードの品質を高められる。

5. **新機能やアーキテクチャの「壁打ち・ブレインストーミング」**
   - 例: 「今度作るアプリのバックエンドについて、SupabaseとFirebaseのメリット・デメリットを比較して」
   - メリット: メインAgentには「コーディング作業」をさせ、Playgroundでは「設計や方針の議論」を行うなど、思考の整理スペースとして活用できる。
