# Skillsとは？ (AIの専門スキル拡張)

Antigravitiにおける「Skills（スキル）」は、AIに特定の複雑なタスクのやり方や、独自のノウハウを教え込むための仕組みです。

## Skillsの「正体」

結論から言うと、Skillsは裏で特殊なAPI通信や魔法のようなプログラムが動いているわけではなく、その正体は**「AIに対する非常に高度で体系化されたプロンプト（指示書）と、必要に応じた補助スクリプトのセット」**です。

## Skillsの役割とメリット（最大の効果）

ユーザーの言葉を借りれば、まさに以下のような役割を果たします。

> 「Skillsはその名の通り、AIにスキルを付与するための『**最大限の知識と絶対的ルールが記載されたガイドブック**』です。」

AIは通常、学習した広範な知識の中から推測して回答しますが、専門的で特殊な最新技術（例：Remotionなど）においては、古い知識や一般的な知識と混ざってしまい、間違ったコード（ハルシネーション）を生成してしまうリスクがあります。

しかし、Skills（ガイドブック）を与えると、AIは以下のように動きます。

1.  **ルールの熟読**: 作業前に必ず `SKILL.md` に記載された絶対ルールや制約を読み込みます。
2.  **推測（ハルシネーション）の排除**: 自分の曖昧な記憶（一般的な推論）に頼るのをやめ、**ガイドブックに書かれた確実なルールとサンプルコードにのみ準拠**して作業を進めるようになります。
3.  **確実な実行**: 結果として、人間が期待した通りの「専門家の品質」で、的確に指示された作業を完了させることができます。

## Skillsのファイル構成例

- **`SKILL.md` (必須)**: AIが読むためのメインの指示書・絶対ルールブック。
- `examples/` (任意): 正しい書き方の見本となるコード。AIはこれを見て書き方のテンプレを学びます。
- `scripts/` (任意): AIがプロジェクト内の環境を調べるためなどに使う小さな補助スクリプト。

## ワークフロー・MCPとの違いのおさらい

- **ワークフロー (`workflows`)**: 「Aをして、次にBをする」という**作業の手順**。
- **Skills (`skills`)**: 「Aをする時は、この専門知識とルールに従う」というAIの脳の拡張（**品質とノウハウ**）。
- **MCP**: AIが外部のデータベースやツールに直接アクセスするための、手足の拡張（**外部との通信手段**）。
