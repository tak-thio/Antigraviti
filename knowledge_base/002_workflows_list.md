# Antigraviti ワークフロー一覧

バイブコーディング（AIとの共同開発）を効率化するために設定されている、便利なワークフローの一覧です。
チャット欄で `/[コマンド名]` を入力するだけで、AIが指定された手順を自動で実行します。

| コマンド        | 役割                           | 説明・使い方の例                                                                                                                                             |
| :-------------- | :----------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`/log`**      | **進捗記録と自動Push**         | 一連の作業が完了した際や1日の終わりに。進捗の要約、日報の作成、`git push`までを全自動で行う。<br>例: `/log 区切り` または `/log 締め`                        |
| **`/snapshot`** | **安全な一時保存**             | 大きなリファクタリングなど危険な変更をする前に、すぐに戻れる「安全地帯」を作る。<br>例: `/snapshot ログイン機能の大幅改修開始`                               |
| **`/review`**   | **AIによる自律コードレビュー** | ひと段落したコードに脆弱性や無駄がないか、AIの視点でチェックしてもらう。<br>例: `/review`                                                                    |
| **`/test`**     | **テスト実行とエラー解析**     | プロジェクトのテストを走らせエラーが出た場合は「原因」と「修正案」まで自動で分析・提示させる。<br>例: `/test`                                                |
| **`/scaffold`** | **ファイルひな形の一括作成**   | 新しい機能を作る際、必要なファイル群と基本コード（インポート等）を一気に作成し、本質的な実装にすぐ入れるようにする。<br>例: `/scaffold 新規ユーザー登録画面` |

※ これらのワークフローの実体（レシピ）は `.agents/workflows/` ディレクトリ内にMarkdown形式で保存されています。内容をカスタマイズしたい場合はこれらのファイルを直接編集してください。
